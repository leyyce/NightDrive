## Controls
| Key         | Command               |
|-------------|-----------------------|
| Arrow Up    | Accelerate            |
| Arrow Down  | Deaccelerate          |
| Arrow Left  | Steer left            |
| Arrow Right | Steer right           |
| F           | Enter/Exit fullscreen |
| L           | Toggle lights         |
